tsc的安装
==========

## 知识点

* typscript编译器tsc的安装
* Playground

## tsc的安装

~~~bash
$ npm install -g typescript
$ tsc -v
$ tsc --help
~~~

### heloworld.ts

~~~js
console.log("Helo TypeScript World.");
~~~

### 代码编译

~~~bash
$ tsc heloworld.ts
~~~

生成 `heloworld.js` 确认
